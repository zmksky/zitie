# ✒️ 字帖生成器 - 使用说明

一款功能强大的在线字帖生成工具，支持自定义文字、字体、格子样式，可直接打印练习。
有唯一编号，编号有惊喜哦
---

## 📋 功能概览

### 🖨️ 打印设置
- **打印张数**：支持1-10页A4纸张
- 系统自动记录打印次数，打印文件名格式：`这是打印的第N个字帖_日期时间`

### 📝 字帖内容

| 功能 | 说明 |
|------|------|
| 文字输入 | 直接输入或粘贴要练习的文字 |
| 去标点 | 一键移除所有标点符号 |
| 内容类型 | 唐诗、宋词、元曲、历代诗词、古文经典等500+首 |
| 随机生成 | 从选定分类中随机抽取诗词填入 |

### 📐 布局设置
- **每行字数**：5-30字可调
- **字体大小**：20-100px可调

### 🔤 字体设置

**三种字体来源：**
1. **免费书法字体**：马善政楷书、站酷小薇体等6款在线字体
2. **自定义字体**：支持上传本地字体文件（.ttf/.otf/.woff/.woff2）
3. **本地字体**：自动检测系统已安装的中文字体

### 🎨 样式设置

**文字样式：**
- 文字颜色（默认黑色）
- 文字透明度（0-100%）

**格子样式：**
- 边框格（仅边框）
- 田字格（十字辅助线）
- 米字格（米字辅助线）
- 格子颜色（默认红色）
- 格子透明度（0-100%）

### ✍️ 练习模式
- **临摹模式**：每行显示范字
- **描红模式**：范字与空行交替，便于对照练习

---

## 🚀 使用方法

### 基础使用
1. 在"字帖内容"区输入要练习的文字
2. 选择字体和格子样式
3. 调整布局参数
4. 点击"打印"按钮

### 使用诗词库
1. 选择"内容类型"（如：唐诗-五言绝句）
2. 点击"随机生成"自动填入诗词
3. 可多次点击获取不同诗词

### 上传自定义字体
1. 点击"📁 上传字体"按钮
2. 选择字体文件（支持多选）
3. 上传的字体出现在"自定义字体"分组中
4. 自动选中并应用

## 💡 使用技巧

1. **自动填充**：设置字数为0时，自动计算填满整页所需字数
2. **文字重复**：当输入文字不足以填满页面时，自动循环重复
3. **描红练习**：选择描红模式，范字行与空白行交替，便于临摹
